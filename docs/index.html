System Architecture  — Three Intelligence Layers

🧠
LAYER 1 — KNN INTELLIGENCE ENGINE
5D Feature Vector · K=8 Nearest Neighbors · 8-Bar Forward Label · Direction-Correct Dual Pool
Short pool: RVOL, ADX, EMA distance, VIX (4D, bearish label). Long pool: RVOL, ADX, EMA reclaim, VIX inverted, HH streak (5D, bullish label). Separate KNN thresholds prevent signal cross-contamination. At K=8 the classifier consults only the 8 most similar historical contexts, producing meaningful signal selectivity versus broad K values that dilute edge into noise.
▼
📡
LAYER 2 — ATLAS v12 LONG ENGINE
EMA21 Pullback Entry · Bullish Engulf Filter · TRENDING_UP Regime Only · Daily SMA20 Macro Gate
Prior bar pulls back to EMA21 (within 0.3%), current bar engulfs above prior high. The ATLAS v12 breakthrough: a daily SMA20 gate blocks all long entries when NQ's daily close is below its 20-day SMA. This single filter removed 7 correction-period losers (Dec 2025, Feb-Mar 2026) while preserving the Apr 2026 momentum cluster. Result: Long PF from 0.923 → 26.075 (s3roGs entity, 50k capital).
▼
⚡
LAYER 3 — OVERLORD SENTINEL
VIX Circuit Breaker (≥35 hard lock) · Strategy Decay Monitor (rolling PF floor 0.50) · Consecutive Loss Lock (6-trade trigger) · Anti-Tilt Fleet Protection · Daily Circuit Breaker ($850)
Four independent risk gates run simultaneously. Any single gate tripping locks all entries regardless of signal quality. VIX spike and consecutive loss gates fire intraday; decay gate monitors rolling performance window. Design principle: protecting capital is more important than capturing any single trade.
